Wir suchen dich!

Wir suchen für dieses Projekt noch weitere Mitarbeiter, die unser freundliches und motiviertes Team unterstützen:

Java Programmierer für den Client

Die Aufgabe des Java Programmierers ist es, den Client mitzuentwickeln.

Java Programmierer für Server

Diese Mitarbeiter kümmern sich um den Server, entwickeln diesen und programmieren all das, was auf dem Server läuft, z.B. dass der Server die GameWorld automatisch generiert und erweitert.

3D Artisten / 3D Modellierer / 3D Animatoren

Ganz wichtig für eine 3D Gamebasis sind natürlich auch die 3D Artisten / 3D Modellierer / 3D Animatoren und Grafiker, die die UI und die 3D Modelle gestalten und animieren, was für den Player sehr wichtig ist. Der erste Eindruck wirkt sich auf das ganze Spiel aus.
